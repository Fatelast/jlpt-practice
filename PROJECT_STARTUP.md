# jlpt-practice 项目启动说明

完整启动文档位于 `启动文档/项目启动.md`。

本项目当前初始化目标：

1. 使用 monorepo 结构。
2. 创建 `apps/miniapp`、`apps/admin-web`、`server`、`packages/shared`、`docs`。
3. 小程序端使用 Taro + Vue3 + TypeScript。
4. 管理后台使用 Vue3 + Vite + TypeScript + Element Plus。
5. 后端使用 NestJS + TypeScript + Prisma。
6. 数据库使用 MySQL。
7. 暂时只创建项目骨架和基础配置，不实现完整业务。
